# html-decoder

> Decodes html

## Usage

```js
const decode = require('path/to/html-decoder')
decode('&lt;span&gt;Hello&lt;/span&gt;')
// => '<span>Hello</span>'
```

## API

### html-decoder(string)

Decodes html

#### string

Type: `string`

HTML string to be decoded.
